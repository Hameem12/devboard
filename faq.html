FAQ s

Question-1: What are the different ways to select an element in the DOM?

Elements can be selected using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets/sets HTML content, innerText gets/sets visible text with formatting, and textContent gets/sets all text including hidden elements, ignoring HTML.

Question-3: What is event delegation in the DOM?

Event delegation is attaching a single event listener to a parent element to handle events on its child elements, improving efficiency.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event triggered on a child element propagates upward to its parent elements unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

Use document.createElement() to create, parent.appendChild() or insertBefore() to add, and parent.removeChild() to remove elements in JavaScript.
